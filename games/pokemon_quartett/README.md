# Pokémon Quartett (Version 1)

Ein konsolenbasiertes Kartenspiel in Python, bei dem mehrere Spieler mit Pokémon-Karten gegeneinander antreten.

---

## 🎮 Spielprinzip

 Jeder Spieler erhält einen Kartenstapel
 Der aktive Spieler wählt eine Kategorie (z. B. Angriff, KP)
 Alle Spieler spielen ihre oberste Karte aus
 Der höchste Wert gewinnt die Runde
 Bei Gleichstand kommt es zu einem Stich
 Der Gewinner erhält alle gespielten Karten

---

## ⚙️ Features

 Mehrspielerfähig (2–6 Spieler)
 Rundenbasiertes Gameplay
 Stich-System (inkl. mehrerer Stichrunden)
 Kartenverteilung per Zufall
 Spielerrotation nach Rundensieg
